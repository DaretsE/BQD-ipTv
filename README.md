# BQDiptv

IPTV-плеер для Android TV и телефона/планшета. Управление пультом (крестовина D-Pad),
настройка плейлистов и EPG со смартфона по локальной сети, телепрограмма с архивом,
избранное, автообновление через GitHub Releases.

Приложение — **только плеер**. Каналы не входят в комплект: пользователь сам
подключает свои легальные плейлисты M3U и источники телепрограммы (XMLTV/EPG).

## Технологии
- Kotlin, Media3/ExoPlayer (HLS), NanoHTTPD (веб-настройка), ZXing (QR-код)
- minSdk 21, targetSdk 34
- Пакет (applicationId): `ru.bqd.iptv` — **менять нельзя**, на нём держится автообновление «поверх»

## Сборка
APK собирается автоматически в GitHub Actions при каждом push в ветку `main`/`master`
(или вручную через вкладку **Actions → Run workflow**). Готовый подписанный
`BQDiptv.apk` и `version.json` публикуются в **Releases**.

Подпись берётся из секретов репозитория: `KEYSTORE_BASE64`, `KEYSTORE_PASSWORD`,
`KEY_ALIAS`, `KEY_PASSWORD`.

Версия задаётся автоматически номером сборки: `versionCode = номер запуска Actions`,
`versionName = 1.0.<номер>`, тег релиза `v<номер>`.

Подробная пошаговая инструкция — в файле **ИНСТРУКЦИЯ.md**.

## Возможности
- Двойная буферизация EPG и иконок (не мигают при обновлении)
- Принудительное обновление плейлиста и EPG (в вебе и в настройках)
- Выбор режима ТВ/Телефон с автоопределением устройства
- Браузер каналов с превью текущей передачи и подписью «осталось …»
- Правая панель телепрограммы, прогресс-бар передачи
- Управление несколькими плейлистами и источниками EPG (добавление, переименование, удаление)
- QR-код для быстрого открытия страницы настройки с телефона
- Адаптивное качество: Авто / Стабильность / Максимум
- Автообновление и откат на предыдущую версию
- Усыпление при сворачивании (освобождает видео-движок, веб-сервер продолжает работать)
