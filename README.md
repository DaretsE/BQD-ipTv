# 📺 Лёгкий Эфир — IPTV-плеер для Android TV

Бесплатный open-source IPTV-плеер по концепции **Zero-Click TV**: включил — и сразу смотришь.
Управление полностью заточено под крестовину пульта (D-Pad).

---

## ЧАСТЬ 1. Как собрать APK БЕЗ ОПЫТА (ничего не устанавливая на компьютер)

Сборка происходит автоматически в облаке GitHub — бесплатно. Вам нужен только браузер.
Время: ~15 минут (из них 5–7 минут машина собирает сама).

### Шаг 1. Создайте аккаунт GitHub
1. Откройте https://github.com и нажмите **Sign up**.
2. Укажите e-mail, придумайте пароль, подтвердите почту.

### Шаг 2. Создайте репозиторий (папку для проекта)
1. Нажмите **+** в правом верхнем углу → **New repository**.
2. Repository name: `legkiy-efir` (любое имя латиницей).
3. Выберите **Public**, остальное не трогайте → **Create repository**.

### Шаг 3. Загрузите файлы проекта
1. Распакуйте полученный архив `LegkiyEfir.zip` в папку на компьютере.
2. На странице репозитория нажмите ссылку **uploading an existing file**
   (или **Add file → Upload files**).
3. Откройте распакованную папку и перетащите в окно браузера **всё её содержимое**
   (папки `app`, `.github`, файлы `build.gradle`, `settings.gradle`, `gradle.properties`, `.gitignore`, `README.md`).
   ⚠️ Важно: перетаскивайте именно содержимое, а не саму папку `LegkiyEfir`.
   ⚠️ macOS: скрытые папки (`.github`) показываются клавишами **Cmd + Shift + .**
4. Внизу нажмите зелёную кнопку **Commit changes**.

### Шаг 4. Дождитесь автоматической сборки
1. Перейдите во вкладку **Actions** (вверху страницы репозитория).
2. Вы увидите запущенный процесс **Build APK** (жёлтый кружок). Подождите 5–10 минут,
   пока кружок не станет зелёной галочкой ✅.
3. Нажмите на завершившуюся сборку → внизу страницы блок **Artifacts** →
   скачайте **LegkiyEfir-APK** (это zip, внутри — готовый `LegkiyEfir.apk`).

> Если сборка не запустилась: вероятно, папка `.github` не загрузилась.
> Решение: в репозитории нажмите **Add file → Create new file**, в имени введите
> `.github/workflows/build.yml`, вставьте содержимое одноимённого файла из архива и нажмите **Commit**.
> Затем во вкладке Actions нажмите **Build APK → Run workflow**.

### Шаг 5. Установите APK на телевизор/приставку
Самый простой способ — флешка:
1. Скопируйте `LegkiyEfir.apk` на USB-флешку, вставьте её в ТВ.
2. На ТВ установите из магазина любой файловый менеджер (например, **File Commander** или **X-plore**).
3. В настройках ТВ разрешите установку из неизвестных источников:
   *Настройки → Приложения → Специальный доступ → Установка неизвестных приложений* → разрешить файловому менеджеру.
4. Откройте APK с флешки → **Установить**.

Способ без флешки — через **Downloader** (есть в магазине на любом Android TV):
1. На GitHub откройте свой репозиторий → справа **Releases → Create a new release** →
   Tag: `v1.0` → перетащите `LegkiyEfir.apk` в поле файлов → **Publish release**.
2. Кликните правой кнопкой по файлу apk в релизе → «Копировать ссылку».
3. В приложении Downloader на ТВ введите эту ссылку — APK скачается и предложит установиться.

## ЧАСТЬ 2. Первый запуск и настройка со смартфона

1. Запустите «Лёгкий Эфир» на ТВ. На экране появится адрес вида **http://192.168.1.50:8765**.
2. Подключите смартфон к **той же Wi-Fi сети**, что и ТВ.
3. Откройте этот адрес в браузере телефона.
4. Вставьте ссылку на ваш `.m3u` плейлист и нажмите «Добавить» — эфир запустится на ТВ мгновенно.

Со смартфона также можно: переименовывать и скрывать плейлисты, задавать ссылку EPG (XMLTV),
менять порядок каналов в Избранном (кнопки ▲▼) и удалять их.

> Это локальный аналог «транзитного токена» из концепта: никаких внешних серверов,
> всё работает напрямую в вашей домашней сети. Запасной вариант — ручной ввод ссылки
> с пульта (ОК на стартовом экране или Настройки в левом меню).

## ЧАСТЬ 3. Управление пультом («Правило крестовины»)

| Кнопка | Действие |
|---|---|
| ▲ / ▼ | Переключение каналов (zapping) |
| **ОК** коротко | Прозрачный список каналов текущей категории |
| **ОК** удержать ~1 сек | Добавить/убрать канал из ⭐ Избранного (анимация звезды) |
| ◀ | Левое меню: плейлисты (листаются ◀ ▶) и категории |
| ▶ | Программа передач текущего канала: ▶ архив (catchup), ● сейчас, ⏰ напоминание |
| Назад | Закрыть любое меню / двойное нажатие — выход |
| MENU | Быстрые настройки: 🎵 аудиодорожка, 💬 субтитры, ⚙ формат кадра |
| CH+ / CH− | Тоже переключение каналов |

Прочие функции из концепта, которые уже работают:
- **Мгновенный старт**: автозапуск при включении ТВ на последнем канале (на части прошивок
  Android 10+ системе нужно один раз разрешить автозапуск приложения).
- **Умная обработка ошибок**: мёртвый поток → сообщение и автопереключение через 3 сек.
- **Избранное** — виртуальный супер-плейлист из всех листов, со смарт-сортировкой по частоте просмотра.
- **Архив**: каналы с тегами `catchup`/`catchup-source` в плейлисте проигрывают прошедшие передачи.
- **Кэш**: логотипы и плейлисты кэшируются, EPG парсится в фоне (поддерживается XMLTV и .gz).
- **Скринсейвер**: пауза дольше 5 минут → чёрный экран с плавающими часами (защита OLED).

## ЧАСТЬ 4. Сборка на своём компьютере (альтернатива, по желанию)

1. Установите **Android Studio**: https://developer.android.com/studio (Next-Next-Finish).
2. Откройте Android Studio → **Open** → выберите распакованную папку проекта.
3. Дождитесь окончания индексации/синхронизации (полоска внизу, первый раз 5–15 минут).
4. Меню **Build → Build App Bundle(s) / APK(s) → Build APK(s)**.
5. По окончании появится уведомление **locate** — APK лежит в
   `app/build/outputs/apk/debug/app-debug.apk`.

## Технические детали

- Язык: Kotlin, плеер: AndroidX **Media3 (ExoPlayer)** с поддержкой HLS.
- Минимальная версия Android: 5.0 (API 21), подходит для Android TV и обычных Android-приставок.
- Веб-настройка: встроенный NanoHTTPD-сервер на порту 8765 (только локальная сеть).
- Лицензия: MIT — используйте и дорабатывайте свободно.

⚠️ Приложение — только плеер. Оно не содержит каналов; используйте собственные легальные
плейлисты (например, от вашего интернет-провайдера или свободные ОТТ-листы).
